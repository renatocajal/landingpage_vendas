📄 Landing Page HTML + CSS

Este repositório contém um modelo de landing page simples, elegante e responsiva, desenvolvido utilizando apenas HTML e CSS puro, sem dependências externas.

🚀 Objetivo

Criar páginas rápidas, leves e fáceis de editar para:

Apresentação pessoal ou profissional
Divulgação de serviços (ex: Reiki, numerologia, terapias)
Captação de leads
Páginas estilo Linktree
🎨 Características
Estrutura 100% em um único arquivo (HTML + CSS embutido)
Design moderno (preto e dourado, se aplicado conforme padrão)
Layout responsivo (adaptável para celular e desktop)
Fácil personalização (cores, textos, links e imagens)
Carregamento rápido (sem bibliotecas externas)
📂 Estrutura

O projeto é composto por:

index.html

Tudo está contido dentro deste único arquivo:

HTML (estrutura)
CSS (estilização)
🛠️ Como usar
Baixe ou clone o repositório
Abra o arquivo index.html no navegador
Edite o conteúdo diretamente no código:
Nome
Descrição
Links
Imagem de perfil
Salve e publique (pode usar GitHub Pages, servidor próprio, etc.)
✏️ Personalização

Você pode alterar facilmente:

Cores no CSS
Textos da página
Botões e links
Imagem de perfil
